# 通信模块全面重构方案

## 一、当前问题分析

### 1.1 ConnectionManager 臃肿 (约400行)
- 大量重复的 switch-case 连接创建逻辑
- 每种协议的代码结构高度相似

### 1.2 目录结构混乱
- `TcpCommon`/`TcpSpecial` 分类标准不清晰
- 4个空目录: `Api/`, `Database/`, `SerialPortCommon/`, `SerialPortSpecial/`
- `IConnection` 目录名语义不明确

### 1.3 命名不统一
- `DLT6452007OverTcp.cs` 缺少 `Driver` 后缀
- 命名风格不一致 (有的用协议名，有的带接口类型)

## 三、核心重构步骤

### 步骤 1: 创建连接工厂接口
在 `Domain/Communication/Factories/` 创建：
```csharp
public interface IConnectionFactory
{
    bool CanHandle(ProtocolType protocolType);
    Task<IConnectionHandle> CreateAsync(ProtocolConfig config, CancellationToken token);
}
```

### 步骤 2: 实现各协议族连接工厂
每个工厂负责一族相关协议，例如 `ModbusConnectionFactory`:
```csharp
public class ModbusConnectionFactory : IConnectionFactory
{
    private static readonly HashSet<ProtocolType> SupportedTypes = [
        ProtocolType.ModbusTcpNet,
        ProtocolType.ModbusRtuOverTcp
    ];

    public bool CanHandle(ProtocolType type) => SupportedTypes.Contains(type);

    public async Task<IConnectionHandle> CreateAsync(ProtocolConfig config, CancellationToken token)
    {
        var netConfig = (NetworkProtocolConfig)config;
        return config.ProtocolType switch
        {
            ProtocolType.ModbusTcpNet => await CreateModbusTcp(netConfig),
            ProtocolType.ModbusRtuOverTcp => await CreateModbusRtuOverTcp(netConfig),
            _ => throw new NotSupportedException()
        };
    }
}
```

### 步骤 3: 简化 ConnectionManager
从 ~400 行简化到 ~50 行：
```csharp
public class ConnectionManager(IEnumerable<IConnectionFactory> factories) : IConnectionManager
{
    private readonly ConcurrentDictionary<string, IConnectionHandle> _connections = [];
    private readonly IEnumerable<IConnectionFactory> _factories = factories;

    public async Task<IConnectionHandle> GetOrCreateConnectionAsync(ProtocolConfig config, CancellationToken token)
    {
        if (_connections.TryGetValue(config.Id, out var existing))
            return existing;

        var factory = _factories.FirstOrDefault(f => f.CanHandle(config.ProtocolType))
            ?? throw new NotSupportedException($"不支持的协议: {config.ProtocolType}");

        var handle = await factory.CreateAsync(config, token);
        return _connections.GetOrAdd(config.Id, handle);
    }
}
```

### 步骤 5: 清理空目录
删除以下空目录：
- `Drivers/Api/`
- `Drivers/Database/`
- `Drivers/SerialPortCommon/`
- `Drivers/SerialPortSpecial/`

### 步骤 6: 重命名领域层目录
- `Domain/Communication/IConnection/` → `Domain/Communication/Abstractions/`

### 步骤 7: 更新依赖注入配置
```csharp
// 注册所有连接工厂

```
